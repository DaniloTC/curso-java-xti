## Introdução Variável

```java
public class Variavel {

    public static void main(String[] args) {
    	
		String nome = "Nome da Variável";
		int idade = 31;
		boolean casado = true;
		
		System.out.println(nome);
		System.out.println(idade);
		System.out.println(casado);
    }
}
```
